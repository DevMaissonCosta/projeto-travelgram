# Travelgram

Projeto de site Travelgram desenvolvido em HTML, CSS e JavaScript.

## Estrutura

- `index.html` - página principal
- `styles/` - arquivos CSS (`header.css`, `main.css`, `footer.css`, etc.)
- `assets/` - imagens e ícones

## Como usar

1. Abra `index.html` no navegador.
2. Edite os arquivos CSS em `styles/` para mudar visual.
3. Adicione conteúdo em HTML conforme necessário.

## Git

1. `git init` (se ainda não inicializou)
2. `git add .`
3. `git commit -m "Adicionar README inicial"`
4. `git remote add origin <URL-do-repositório>`
5. `git push -u origin main`
